Es wird eine Möglichkeit geschaffen, auf die Dauer von Services individuell einzuwirken. Dafür wird in der Servicelis te (links im Planungscockpit) ein neues Feld für den Auf-/Abschlag in Einheit “MIN" (Minuten) eingefügt sowie eine neue Funktion zum Ändern dieses Werts. Dabei sollen sowohl positive(Zeit-Aufschlag) als auch negative Eingaben (Zeit-Aufschlag) erlaubt sein.
